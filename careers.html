Our Careers

Our job openings close in:

Website Manger

Responsible for overseeing the development, maintenance, and performance of the website. This position requires technical expertise in website design, development, and management, as well as strong communication skills and a keen understanding of the target audience.

Front-End Programmer

Responsible for creating the visual design and layout of a website. They use tools such as Adobe Photoshop, Illustrator, or Sketch to create wireframes and mockups, and then use HTML, CSS, and JavaScript to bring the design to life.

Backend Developer

Responsible for building the server-side functionality of the website. Use programming languages such as PHP, Ruby on Rails, or Java to create features such as user authentication, database management, or server-side scripting.

SEO Specialist

Responsible for optimizing the website's content and structure to improve its visibility in search engine results. Use techniques such as keyword research, on-page optimization, and link building to improve the website's ranking.

UX Designer

Responsible for creating the user experience of the website. Use research, prototyping, and testing to create a design that is intuitive, user-friendly, and meets the needs of our audience.
